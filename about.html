SecureKey란?

SecureKey는 암호학적으로 안전한 비밀번호 생성과 비밀번호 강도 분석을 제공하는 무료 온라인 도구입니다.

단순히 비밀번호를 만들어주는 것을 넘어, 왜 해당 비밀번호가 강하거나 약한지 8가지 기준으로 분석하고 엔트로피(정보량)까지 계산해 사용자가 보안을 이해할 수 있도록 돕습니다.

주요 기능

🔑

비밀번호 생성기

대문자·소문자·숫자·특수문자 조합을 자유롭게 선택하고, 4~64자 길이의 비밀번호를 즉시 생성합니다. 최근 8개의 생성 기록도 확인할 수 있습니다.

🛡

비밀번호 강도 체크

기존 비밀번호를 입력하면 길이, 문자 다양성, 반복·연속 패턴, 흔한 비밀번호 여부 등 8가지 기준으로 점수를 산출합니다. 엔트로피(bits)도 함께 표시됩니다.

보안 원칙

SecureKey는 다음 원칙을 지킵니다.

🎲

암호학적 난수 사용

브라우저의 crypto.getRandomValues() API를 사용합니다. 예측 가능한 Math.random()이 아닌 진정한 암호학적 난수를 기반으로 비밀번호를 생성합니다.

🔒

완전한 클라이언트 처리

입력하거나 생성한 비밀번호는 서버로 전송되지 않습니다. 모든 처리는 사용자의 브라우저 안에서만 이루어지므로 네트워크를 통한 정보 유출이 원천 차단됩니다.

이용 방법

별도의 설치나 회원가입 없이 메인 페이지를 방문하면 즉시 사용할 수 있습니다. 로그인도, 앱 설치도 필요하지 않습니다.
