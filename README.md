# SecureKey 비밀번호 생성기

브라우저에서 바로 실행되는 안전한 비밀번호 생성기 & 강도 분석기입니다.

## 주요 기능

### 비밀번호 생성기
- 대문자, 소문자, 숫자, 특수문자 조합 선택
- 길이 설정 (4~64자)
- 실시간 강도 표시
- 최근 생성 기록 (최대 8개)
- 클립보드 복사

### 비밀번호 강도 체크
- 8가지 기준으로 점수 산출 (길이, 문자 종류, 반복/연속 패턴, 흔한 비밀번호 여부 등)
- 엔트로피(비트) 계산
- 취약점 체크리스트 시각화

## 보안

- `crypto.getRandomValues()` 기반 암호학적 난수 사용
- 모든 처리는 클라이언트(브라우저)에서만 이루어지며, 입력한 비밀번호를 서버로 전송하지 않습니다.

## 실행 방법

별도의 설치나 빌드 없이 `index.html`을 브라우저로 열면 바로 사용할 수 있습니다.
